## 🧠 How It Works — Adaptive Learning

Movable is built on proven learning science:

```
Zipf's Law (word frequency)
        ↓
Vocabulary Engine (SRS SM-2)
        ↓
Adaptive AI Daily Plan
        ↓
Practice Modules: Read → Hear → Speak → Write
        ↓
Progress Tracking (CEFR A0 → B2)
        ↓
Next personalised plan
```

| Principle | How Movable applies it |
|-----------|----------------------|
| **Zipf's Law** | Most frequent Dutch words first — fastest path to fluency |
| **Spaced Repetition (SM-2)** | Review words exactly when you're about to forget them |
| **Krashen i+1** | Every exercise is just one step above your current level |
| **Lexical Ladder** | 98% known words + 2–5 new ones in every text |
